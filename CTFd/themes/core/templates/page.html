How to play

  • NullByte CTF is a Capture the flag event hosted by Turing category of MIT Manipal's TechTatva
  • The challenges will be live on 10th and 11th of October.
  • This is a jeapordy based CTF with categories like Forensics, PWN, Cryptography, Web, Reversing etc.
  • Solve the challenge with the given statement and files to get a flag.
  • A flag is a string of the format NullByteCTF{string}.
  • Submission of a correct flag gives the mentioned points.
  • Hints will be added if required.
  • User with the most points is the winner.
